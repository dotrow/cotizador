== Description ==

Ajax Gallery es plugin para wordpress que permite mostrar imagenes de Picasa Google en cualquier post/pagina.
Utilizando la API de Picasa Google y utlizando como base jQuery, Ajax Gallery es el mas elegante y rapido plugin para mostrar imagenes en nuestro blog

*Caracteristicas*

1. Utiliza Thickbox
2. Soporta multiples galerias
3. Se puede editar una galeria sin modificar el codigo fuente
4. Muestra paginaci�n
5. Es Personalizable
6. Suporta mutiples albums
7. Soporta thumbnails

== Frequently Asked Questions ==

= He configurado todo, Por que no veo nada? =

Debido a las politicas de Picasa Google, por el momento solo se muestran los albums publicos, no funciona con privados

= Por que el album que especifique no se muestra? =

El album especificado en las configuraciones de cada galeria debera ser sin puntos, comas o espacios intermedios. 
Ej.  KDE 2.0 ser�a KDE20

= Como incrustar una galeria en un post/pagina? =

Cuando usted cree una nueva galeria se generara un codigo, el cual debera copiar y pegar en el post/pagina que desee

= Como modifico una galeria? =

* Ir al menu Ajax Gallery
* En la lista de galerias escojer la que desee modificar
* Por ultimo personalizarla

= Como actualizo el plugin? =

* Primero tienes que desactivar el plugin que tienes en tu wordpress
* Eliminar el directorio de ajaxgallery que se encuetra en la carpeta de plugins
* Descargar la ultima version de Ajax Gallery
* Realizar los pasos de una nueva instalacion nueva
* Activar el plugin

== Installation ==

ES:

1. Descargue el plugin
2. Descomprima el archivo ajaxgallery.zip o ajaxgallery.rar
3. Copie la carpeta ajaxgallery en la carpeta wp-content/plugins de wordpress
4. Inicie sesion en su WordPress.
5. Haga click en Plugins desde el menu principal.
6. Busque el nombre del plugin - ajaxGallery, y de click en activar
7. Vaya a opciones del menu principal y modifique los datos de la galeria como usted quiera
8. Cuando edite un post o cree una pagina escriba en el editor de texto de wordpress escriba [ajax_gallery]
9. Publique su pagina o post
10. Abra la pagina desde cualquier explorador

EN:

1. Download the ajaxgallery for WordPress archive
2. Unzip the ajaxgallery.zip/.rar file
3. Place the folder ajaxgallery into your wp-content/plugins folder
4. Log in to your WordPress dashboard
5. Open the Plugins menu
6. Locate ajaxgallery and click activate
7. Take a look at the main-menu options and change them as you see fit
8. When you write a post or page, just enter [ajax_gallery] in the text editor to generate a gallery in that post/page
9. Publish the post/page
10. Open the post/page in any browser
